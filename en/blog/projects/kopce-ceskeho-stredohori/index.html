• Kopce Českého Středohoří

  • 2021/05/23

I created this project because I like to go out in my free time . And preferably on the hills. České Středohoří is beautiful, so I decided on this area.

The first idea was to create an interactive map where I would check the hills I was on. I created this quite quickly, so I started adding various features. I created a fairly large database of hills with their information (location, height, prominence, isolation, material, basin, etc.) . So a feature has been added where you can click on a hill to see various information about the hill.

Everyone probably knows, as programmers, that if we don't know what to create for a project, it's bad . So I wondered in the editor and modified the code to look good and be effective (Which I probably didn't do very well). And it occurred to me that morning. The user would definitely welcome a feature that would display the area of already-conquered hills . So I started looking. Fortunately, api.mapy.cz offers geometric functions such as lines and polygons . Now all that remained was to find out how the hell to choose points that are more or less the border of all points. The Convex Hull algorithm helped me with this. I know it's not perfect, but it performs nicely, so shut up.

The last feature I added is search . Because who would love to look for a hill on the map manually . Searching wasn't as big a problem as I thought. I wrote a script that solves it quite well. For whisps I used JQuery UI library , where autocomplete is included. The biggest problem was probably the Czech characters :D, fortunately, colleagues from JQuery UI took this into account and created character mapping . So minimal hassle.
