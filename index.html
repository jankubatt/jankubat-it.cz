About Me

Hey there! Welcome to my web portfolio! I'm Jan Kubat, a 19-year-old full-stack web developer who's been tinkering with code since 2017. I'm super excited to show you what I've been up to lately!

I've worked on some pretty cool projects recently, like Kopcuj, BakaBot, and MorpheusSystems. Each project has challenged me in different ways and helped me grow as a developer. Right now, I'm working at Crudex, where I get to work with an amazing team of talented folks.

I'm all about keeping things simple and organized. I believe that the best code is code that's easy to read and maintain. And, let's be real, who doesn't love a minimalist design? ;)

One of my favorite things about being a web developer is that there's always something new to learn. I'm a fast learner and always eager to take on new challenges. Plus, I'm pretty precise when it comes to coding - I like to make sure everything is just right.

Thanks for checking out my portfolio! I hope we get a chance to work together soon!
